This game was implemented with the [CodeDraw library](https://github.com/Krassnig/CodeDraw).

```bash
# install java
brew install openjdk

# run the game
javac -cp ./lib/CodeDraw.jar ./src/Mastermind.java -d ./out
java -cp ./out:./lib/CodeDraw.jar Mastermind
```
