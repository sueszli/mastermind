a little demo for the [codedraw library](https://github.com/Krassnig/CodeDraw)

```bash
javac -cp ./lib/CodeDraw.jar ./src/Mastermind.java -d ./out
java -cp ./out:./lib/CodeDraw.jar Mastermind
```
